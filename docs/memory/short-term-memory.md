# Short-Term Memory

## 2026-06-13 Latest Strategy

The current product strategy is user-zero first: Re_Call is being built for Adam as the first user. Adam's taste and natural reaction are the success bar for wording, components, functionality, API calls, font size, color scheme, recommendation engine, ontology, knowledge graph, and every other detail. Anything Adam does not understand when communicating with an agent may harm the project. Features for others are allowed only when Adam intentionally chooses them; the app is always designed solely for Adam first.
